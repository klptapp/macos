# KLPT Beta Welcome Guide

A minimal static site that welcomes new beta testers to KLPT.

## Preview locally

```sh
cd docs/welcome
python3 -m http.server 8000
# open http://localhost:8000
```

## Files

- `index.html` — single-page guide
- `styles.css` — premium minimalist macOS dark theme
- `assets/` — UI screenshot crops used throughout the page
